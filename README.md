# Data-Analysis-
 Data Analysis using Pandas and NumPy
Project Overview
This project performs intermediate-level data analysis on a dirty employee dataset using Python, Pandas, and NumPy.

The dataset contains
- Outliers
- Map
- loc
- iloc

Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

The main goal of this project is to clean the dataset and perform meaningful analysis using Pandas operations.
